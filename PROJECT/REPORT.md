# Lecture 7 Report – Code You Never Write

## Project 1 – Money Detective

### Problem

Analyze transaction history to find recurring charges and spending patterns.

### AI Tool Used

Claude

### Verification

Manually calculated transaction totals and compared them with the script output.

### Result

The script identified recurring transactions and calculated total spending correctly.

### Learning

AI can automate financial analysis, but outputs should always be verified.

---

## Overall Reflection

This assignment demonstrated the Code You Never Write approach. Instead of writing code manually, I acted as a client by describing problems, reviewing AI-generated solutions, verifying outputs, and documenting the results. The most important lesson was that AI-generated outputs should never be trusted without verification.
