## Overview

This repository contains four AI-assisted projects completed as part of the Code You Never Write assignment. The objective was to approach problems as a client by describing requirements to AI, reviewing generated solutions, verifying results, and documenting the entire process.

Throughout these projects, AI was used to generate Python scripts while manual verification was performed to ensure accuracy and reliability.

## AI Tools Used

* ChatGPT
* Python
* Visual Studio Code (VS Code)

## Project 1 – Money Detective

This project focused on analyzing transaction data to identify recurring charges, duplicate payments, and overall spending patterns. A transaction file was provided as input, and AI generated a script to process the data and summarize spending activity. The results were manually verified by checking transaction totals and recurring entries.

## Project 2 – What's My Grade, Really

This project involved calculating a student's current grade based on weighted assessment categories such as assignments, quizzes, and exams. The generated script applied the grading rules, calculated the current standing, and estimated the score required on the final exam to achieve a target grade. Manual calculations were used to verify the averages and weighted contributions.

## Project 3 – The Books Don't Match

The purpose of this project was to reconcile payment records against an expected total amount. AI generated a script that calculated the amount received, compared it with the expected value, and identified any missing funds. The output was verified by manually summing the payments and comparing the results.

## Project 4 – Organize the Mess

This project focused on analyzing a folder containing different file types and potential duplicates. A safe approach was followed by creating a backup before analysis. The generated script grouped files by type, highlighted possible duplicate files, and produced a summary without modifying any original data. Results were verified through manual inspection of the folder contents.

## Key Learning Outcomes

Through these projects, I learned how AI can be used to generate useful solutions for real-world problems while reducing the amount of manual coding required. At the same time, I learned the importance of verification, testing, and responsible use of AI-generated outputs. The assignment demonstrated that successful use of AI requires clear problem descriptions, careful review of generated solutions, and validation of results before they are trusted.

## Conclusion

The Code You Never Write approach emphasizes problem-solving, verification, and critical thinking rather than writing every line of code manually. These projects showed how AI can act as a powerful assistant for software development while still requiring human judgment to ensure correctness and reliability.
