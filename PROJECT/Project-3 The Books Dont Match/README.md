# Project 3 – The Books Don't Match

## Problem

The goal of this project was to compare an expected payment total with the actual received payments and identify any missing amount.

## AI Tool Used

Claude

## Verification Method

I manually added all payment amounts and compared the result with the script output.

## Result

The script calculated a received total of 450 and identified a missing amount of 50 from the expected total of 500.

## What I Learned

AI can help automate financial reconciliation tasks, but results should always be verified using known totals.
