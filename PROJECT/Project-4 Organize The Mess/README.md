# Project 4 – Organize the Mess

## Problem

The goal was to analyze a messy folder, group files by type, and identify possible duplicate files safely.

## AI Tool Used

ChatGPT

## Verification Method

I manually checked the folder contents and compared them with the script output.

## Result

The script successfully grouped files by type and identified possible duplicate files without modifying any original files.

## What I Learned

AI can help organize files, but file operations should always be tested on a backup copy before making changes.
