# Project 2 – What's My Grade Really

## Problem

The goal of this project was to calculate a student's actual grade using specific grading rules and determine the score needed on the final exam to achieve a target grade.

## AI Tool Used

ChatGPT

## Verification Method

I manually calculated the assignment average and compared it with the value used by the script. The results matched.

## Result

The script calculated the current grade and estimated the score required on the final exam to achieve the target grade.

## What Worked

The grading formula correctly applied category weights and produced consistent results.

## Problems Faced

I needed to ensure that the grading weights added up correctly and were applied to the correct categories.

## What I Learned

AI can generate custom grade calculators based on unique grading policies that standard grade apps may not support.
