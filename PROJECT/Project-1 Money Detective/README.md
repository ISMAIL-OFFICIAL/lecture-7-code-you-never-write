# Project 1 – Money Detective

## Problem

I wanted to analyze transaction history to identify recurring charges, duplicate payments, and spending patterns.

## AI Tool Used

Claude

## Prompt Used

I have transaction data with date, description and amount.

Write a Python script that:

1. Finds recurring subscriptions.
2. Finds duplicate payments.
3. Calculates total spending.
4. Creates a spending summary.

Explain the code in simple English.

## Verification

I manually calculated the total spending by adding all transaction amounts and compared it with the script output.

## Result

The script successfully identified recurring charges such as Netflix, Spotify, and KFC and calculated the total spending correctly.

## What I Learned

AI can quickly generate useful analysis scripts, but results should always be verified against known values before trusting them.
