## Overview

This repository contains four AI-assisted Python projects completed as part of the **Code You Never Write** assignment. The purpose of these projects was to solve practical problems by describing requirements to an AI assistant rather than writing every line of code manually. Throughout the assignment, I primarily used **Claude** to generate Python scripts, while using **ChatGPT** for additional guidance and explanations when needed. All scripts were tested in Visual Studio Code, manually verified, and documented.

## AI Tools Used

* Claude AI (Primary)
* ChatGPT (Secondary)
* Python
* Visual Studio Code (VS Code)

## Project 1 – Money Detective

This project analyzes transaction data to identify recurring charges, duplicate payments, and spending patterns. A transaction dataset was used as input, and an AI-generated Python script processed the data to calculate total spending and identify repeated transactions. The results were manually verified to ensure that the calculated totals and recurring entries were accurate.

## Project 2 – What's My Grade, Really

This project calculates a student's academic performance using weighted grading criteria. The script computes assignment and quiz averages, applies category weightages, determines the current overall grade, and estimates the score required on the final exam to achieve a target grade. Manual calculations were performed to verify the averages and weighted results produced by the script.

## Project 3 – The Books Don't Match

This project focuses on payment reconciliation by comparing an expected total amount with actual payments received. The AI-generated script calculates the received total, identifies any missing amount, and presents the results clearly. The calculations were verified manually by adding all recorded payments and comparing them with the expected value.

## Project 4 – Organize the Mess

This project demonstrates safe file organization practices. A sample folder containing different file types and possible duplicate files was analyzed using an AI-generated Python script. The script grouped files by type, highlighted potential duplicate files, and produced a summary report without modifying any files. A backup folder was created before analysis, and the results were manually verified by checking the folder contents.

## Key Learning Outcomes

These projects demonstrated how AI can assist in solving programming problems efficiently through prompt engineering and automation. More importantly, they emphasized the importance of verifying AI-generated outputs, testing solutions carefully, documenting the development process, and using AI responsibly rather than relying on AI-generated code without validation.

## Conclusion

The **Code You Never Write** approach shifts the focus from manually writing every line of code to understanding problems, communicating requirements effectively, evaluating AI-generated solutions, and validating results. This assignment strengthened my understanding of AI-assisted software development while reinforcing the importance of human judgment, verification, and critical thinking in programming.
